## Nature / Science

### Style Characteristics
- **Tone**: Authoritative, accessible, impactful
- **Length**: Concise (typically 1-2 pages)
- **Focus**: Broad significance and impact

### Report Format
```
To the Editor:

We were pleased to review this manuscript...

OVERALL SUMMARY
The manuscript by [Authors] reports...

Major Concerns
1. [Clear, direct statement of issue]

Minor Concerns
1. [Suggestion or minor issue]

AI Reference Comments
[If applicable - clearly marked as non-binding]

Recommendation
[Clear recommendation with brief justification]
```

### Language Notes
- Use active voice when possible
- Emphasize significance and broad appeal
- Be direct but constructive
- Avoid excessive technical jargon

## Advanced Materials / Wiley

### Style Characteristics
- **Tone**: Technical, detailed, thorough
- **Length**: Comprehensive (2-4 pages)
- **Focus**: Technical depth and methodological rigor

### Report Format
```
Dear Editor:

OVERALL SUMMARY
The work presents...

Technical Evaluation
[Detailed assessment of methodology]

Major Issues
1. [Detailed technical concern]
   Specific location: Section X
   Current text: "..."
   Issue: ...
   Suggestion: ...

Minor Issues
1. [Specific suggestion]

AI Reference Comments
[If applicable]

Recommendation
[Technical justification]
```

### Language Notes
- Use precise technical terminology
- Provide detailed evidence and references to specific sections
- Be thorough in explaining technical concerns
- Include specific suggestions for improvement

## ACS

### Style Characteristics
- **Tone**: Technical, systematic, structured
- **Length**: Balanced (2-3 pages)
- **Focus**: Technical rigor and experimental validation

### Report Format
```
Dear Editor,

REVIEWER COMMENTS

Manuscript ID: [if applicable]
The manuscript entitled "[Title]" by [Authors] has been reviewed...

SUMMARY
The paper reports on...

MAJOR COMMENTS
1. [Technical issue with specific location]
   The authors state "..." (Section X.X, Page X)
   However, this presents a concern because...
   I recommend that...

MINOR COMMENTS
1. [Suggestion or clarification]

AI Reference Comments
[If applicable]

RECOMMENDATION
[Detailed recommendation with technical justification]
```

### Language Notes
- Be systematic and methodical
- Focus on experimental design and validation
- Provide numbered references to specific sections
- Balance criticism with recognition of technical merits

## IEEE

### Style Characteristics
- **Tone**: Technical, precise, engineering-focused
- **Length**: Comprehensive (2-3 pages)
- **Focus**: Technical accuracy and innovation

### Report Format
```
Dear Editor and Reviewers:

MANUSCRIPT EVALUATION
Paper ID: [if applicable]
Title: [Full title]

SUMMARY
The paper presents...

TECHNICAL REVIEW
[Numbered sections matching paper structure]

1. Originality and Innovation
2. Technical Correctness
3. Experimental Validation
4. Clarity of Presentation

MAJOR COMMENTS
[Clearly numbered technical issues]

MINOR COMMENTS
[Suggestions for improvement]

AI Reference Comments
[If applicable]

RECOMMENDATION
[Detailed technical recommendation]
```

### Language Notes
- Use precise technical language
- Reference specific equations, figures, and sections
- Focus on technical correctness
- Be systematic and organized

## Elsevier

### Style Characteristics
- **Tone**: Formal, constructive, scholarly
- **Length**: Balanced (2-3 pages)
- **Focus**: Constructive feedback and scholarly standards

### Report Format
```
Dear Editor,

REVIEWER COMMENTS

The manuscript entitled "[Title]" by [Authors] has been 
reviewed according to the journal's standards...

GENERAL COMMENTS
[Overall assessment]

SPECIFIC COMMENTS

Comment 1: [Major issue]
Location: Section X.X, Page X
The authors state that "..."
However, this is problematic because...
I suggest that...

Comment 2: [Additional major/minor issue]
...

AI Reference Comments
[If applicable]

CONCLUSION
Based on the above review...

RECOMMENDATION
[Accept / Minor Revision / Major Revision / Reject]
```

### Language Notes
- Be constructive and helpful
- Balance criticism with recognition of strengths
- Use scholarly language
- Provide actionable suggestions

## Springer

### Style Characteristics
- **Tone**: Academic, formal, comprehensive
- **Length**: Standard (2-3 pages)
- **Focus**: Scholarly standards and methodological rigor

### Report Format
```
Dear Editor,

REVIEWER REPORT

Manuscript Title: [Full title]
Authors: [List]

SUMMARY
[A brief overview of the manuscript]

REVIEWER COMMENTS

1. Originality and Significance
[Assessment of novelty and importance]

2. Methodology
[Evaluation of research design and methods]

3. Results and Discussion
[Assessment of findings and interpretations]

4. Presentation
[Writing quality, figures, tables]

MAJOR POINTS
[Numbered list of major concerns]

MINOR POINTS
[Numbered list of minor suggestions]

AI Reference Comments
[If applicable]

RECOMMENDATION
[Detailed recommendation with justification]
```

### Language Notes
- Maintain academic formality throughout
- Be thorough but concise
- Address all aspects systematically
- Provide clear justification for comments

## General Guidelines

### Universal Principles

1. **Be Constructive**
   - Acknowledge strengths before criticism
   - Provide actionable suggestions
   - Focus on improving the manuscript

2. **Be Specific**
   - Reference specific sections, figures, or equations
   - Quote relevant text when addressing issues
   - Provide concrete examples

3. **Be Objective**
   - Base comments on evidence and standards
   - Avoid personal opinions or preferences
   - Distinguish between preferences and requirements

4. **Be Proportionate**
   - Reserve critical language for significant issues
   - Match comment severity to actual problems
   - Consider the scope of the journal

### Common Phrases by Severity

#### Fatal/Critical Issues
```
"The manuscript cannot be accepted because..."
"This fundamental flaw invalidates the main conclusion..."
"The data does not support the claims presented..."
```

#### Major Issues
```
"We have concerns regarding..."
"It would be necessary to address..."
"The manuscript would benefit from..."
"We suggest that the authors reconsider..."
```

#### Minor Issues
```
"It might be helpful to..."
"Consider revising..."
"A minor suggestion would be..."
"This could be clarified by..."
```

#### Editorial Issues
```
"Please check the formatting..."
"Consistency in terminology is recommended..."
"Reference style should be verified..."
```
